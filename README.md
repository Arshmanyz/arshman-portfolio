# Arshman Portfolio

Arshman 的个人产品展示网站首版。

网站已配置 Sanity 项目 `j5cki43u`，线上环境会优先读取 Sanity 的产品和资源内容，本地文件预览继续使用内置内容。

## 本地预览

直接打开 `index.html`，或在此目录启动任意静态文件服务器。

## 后续替换

- 提供真实联系邮箱后，接入联系入口和订阅服务。
- 产品上线后，将状态按钮替换为产品下载或体验链接。
- 有真实产品截图后，替换页面中的界面概念预览。
- PDF 完成后放入 `assets/documents` 并更新资源链接。

## 内容后台

Sanity Studio 位于 `studio` 目录，使用 `production` dataset。后台包含：

- 产品
- PDF 资源
- 网站设置
